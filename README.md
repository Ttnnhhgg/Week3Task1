# Week3Task1
Book API - ASP.NET Core Web API

Простое Web API для управления авторами и книгами с временным хранилищем в памяти.

Функциональность
- CRUD операции для авторов и книг
- Валидация данных
- REST API
- Swagger UI для тестирования

Технологии
- ASP.NET Core 8
- Swagger/OpenAPI
- In-Memory хранилище
